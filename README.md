## Overview
A Scala library made to interact with Discord's API for bots. The library is strictly designed for botsâ€”any other forms of interaction that may violate Discord's developer policy are not supported nor allowed.

## Usage
To provide a quick, mostly-synchronous, and simpler approach to making and running bots on Discord. To further expand on the permissiveness of interacting with Discord's API to create useful software for users.
